uMessages Introduction
====================

This documentation covers 1.0 release of django-umessages application. A Django
application that supplies you with iPhone like messaging system for your
users.

Why uMessages?
====================
A **Messaging** system between users that either get's displayed as conversations
 (iPhone like) or sorted per subject (Gmail).

Help out
========

Found a bug in uMessages? File an issue at Github. Have an improvement? Fork it
and add it, or if you can't code it, contact us to do it.
